### syntactic category (구문적 범주)   
e.g. in English : noun, verb, adjective, ....
	in programming languages : identifier, number, operator, ...
<br>

token은 이름과 값(optional)이 한 쌍을 이뤄 구조를 이룸.   
e.g. identifier A     
	token name : identifier   
	token value : A
